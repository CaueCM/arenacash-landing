# arenacash-landing

Landing page fake da **ArenaCash** — material do exercício de GitHub do curso *Claude para PMs · Do Chat ao Código*.

Você vai usar esse repo pra praticar o fluxo completo:

1. **Fork** desse repo pra sua conta
2. **Clone** o seu fork no Claude Code
3. **Edite** o `index.html` (vamos ajustar copy do botão e do subtítulo)
4. **Commit** e **push**
5. **Abra um Pull Request** no repo original

Tudo em conversa com o Claude. Sem digitar comando git.

## O que tem aqui

- `index.html` — a landing page completa, em 1 arquivo só.

Pra ver como ficou no navegador, é só abrir o `index.html` direto.

## O que você vai mudar

Durante o exercício, a CMO da ArenaCash pediu pra testar dois novos copies:

| Antes | Depois |
|---|---|
| `O cash da nova economia.` | `Sua conta digital em 90 segundos.` |
| `Abra sua conta` | `Comece agora` |

Você vai fazer essa mudança **pelo Claude Code** — sem abrir editor de código, sem digitar HTML.
